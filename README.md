# Formulario con React y Styled-Components.


### Tecnologías

* React.
* Styled-Components.
* Css.
* Javascript.
* Font Awesome.

## Instalación

1. Descargar el proyecto del repositorio.
2. Instalar las dependencias de la aplicación con el comando 'npm install'.
3. Ejecutar la aplicación con el comando 'npm start'.
